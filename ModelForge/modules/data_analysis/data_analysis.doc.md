# **data_analysis**

В данной папке реализованы модули для анализа различных табличных данных, подбора и обучения модели ML.

## *data_statistics.py*

Модуль с функциями для вывода статистик по предоставленным данным.

## *gboost_models.py*

Модуль предоставляет доступный интерфейс для обучения моделей и подбора оптимальных моделей случайного леса, реализованных библиотекой xgboost. Подбор параметров осуществляется с помощью GridSearchCV по предоставленной сетке параметров.

## *regression_task.py* (in progress)

В разработке находится интерфейс для обучения моделей под регрессионные задачи.

## *show_result_plot.py*

Набор универсальных функция для формирования итоговых графиков, промежуточных графиков, в процессе работы скрипта. Сохранения графиков в папку plots

## *trend_season_analysis.py*

Функции для анализа табличных данных, сущность которых - временные ряды.
