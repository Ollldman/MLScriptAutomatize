# **integration**

В данной папке созданы функции для управления интеграционными процессами с обученной моделью, удаленный или запланированный запуск скрипта. Установка API взаимодействия. Запись результатов эксперимента в базу данных.

## *api_integration.py*
Отработка базового скрипта проекта, при получении PipelineRequest объекта с корректно заполненными полями. Возвращает качество результата и папку с итоговым отчетом.

## *cron_executor.py*

Базовый pipeline для быстрой отработки решений по запланированному по графику запуску.

## *db_storage.py*

Хранилище-объект базы данных и функция для записи результатов выполнения скрипта в базу данных.

## *logging_pipeline.py*
Функции для универсального логирования общего pipeline.
