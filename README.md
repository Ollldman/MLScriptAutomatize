# **Скрипт для автоматизации рутинных задач инженера машинного обучения**
## *1. Задачи, которые необходимо решить.*
1. Автоматизация загрузки данных из различных источников:

    - csv / excel, +
    - API (Kaggle) + 
    - Локальная база данных (Реляционная SQL СУБД на примере PostgreSQL), +
    - Не реляционная база данных (NoSQL - MongoDB),
    - Документация по модулям загрузки данных. +

2. Валидация данных при загрузке

    - Проверка на дубликаты, +
    - Проверка пропусков и корректности данных, +
    - Выявление и обработка выбросов, +
    - Логирование результатов валидации, +
    - Документация по модулям валидации данных. +

3. Очистка данных +
4. Анализ данных ->
    - Подсчет статистик: среднее, медиана, мода, стандартное
отклонение,
    - Анализ трендов и сезонности во временных рядах,
    - Выявление аномалий (выбросов),
    - Построение и обучение базовой ML-модели (регрессия или
классификация),
    - Вывод метрик качества модели.
5. Отчетность
Формирование отчета выполним по этапам CRISP-DM, в таком случае итоговый отчет будет генерироваться пошагово в процессе выполнения программного кода:
### Шесть этапов CRISP-DM:

| Этап | Описание |
|------|--------|
| **1. Business Understanding**<br>Понимание бизнеса | Определение целей проекта, ключевых метрик, ограничений, ресурсов. Что мы хотим достичь? |
| **2. Data Understanding**<br>Понимание данных | Исследование данных: загрузка, просмотр, поиск пропусков, выбросов, распределений. Первые визуализации. |
| **3. Data Preparation**<br>Подготовка данных | Очистка: обработка NaN, дубликатов, выбросов; кодирование категориальных признаков; нормализация; создание новых фич. |
| **4. Modeling**<br>Моделирование | Выбор алгоритмов, обучение моделей, подбор гиперпараметров (GridSearch), кросс-валидация. |
| **5. Evaluation**<br>Оценка модели | Анализ метрик качества (R², MAE, F1, AUC и т.д.), проверка на тестовых данных, интерпретируемость. |
| **6. Deployment**<br>Внедрение | Интеграция модели в production, API, отчетность, автоматизация. |

6. Интеграция с бизнесс-процессами
7. Документация и тестирование. +-
